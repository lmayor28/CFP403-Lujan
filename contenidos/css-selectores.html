Selectores CSS Fundamentales

¿Qué son los Selectores CSS?

Los selectores CSS son patrones que se utilizan para **seleccionar y apuntar a elementos HTML** que quieres estilizar. Sin selectores, el CSS no sabría a qué parte de tu página aplicar un estilo determinado.

Son la parte inicial de una regla CSS (antes de las llaves {}).

Selectores Comunes

Selector de Elemento (Type Selector)

Selecciona todos los elementos de un tipo HTML específico (ej: todos los párrafos <p>, todos los encabezados <h1>, etc.).

p {
  color: blue;
}

h1 {
  text-align: center;
}

Selector de ID (ID Selector)

Selecciona un **único** elemento con un id específico. Los IDs deben ser únicos en una página HTML.

Se denota con el símbolo de almohadilla # seguido del valor del ID.

#miElementoUnico {
  border: 1px solid red;
}

HTML:

<div id="miElementoUnico">Este div tiene un ID.</div>

Selector de Clase (Class Selector)

Selecciona todos los elementos que tienen un atributo class específico. Las clases pueden ser reutilizadas en múltiples elementos.

Se denota con un punto . seguido del nombre de la clase.

.destacado {
  font-weight: bold;
  color: green;
}

HTML:

<p class="destacado">Este párrafo está destacado.</p>
<span class="destacado">Este span también.</span>

Selector Descendiente (Descendant Selector)

Selecciona un elemento que es descendiente de otro elemento (puede estar anidado a cualquier nivel dentro del primer elemento).

Se denota simplemente con un **espacio** entre los selectores.

div p {
  /* Selecciona todos los párrafos que están dentro de un div */
  margin-bottom: 10px;
}

HTML:

<div>
  <p>Este párrafo será afectado.</p>
  <section>
    <p>Este párrafo también será afectado.</p> <!-- Es descendiente del div -->
  </section>
</div>
<p>Este párrafo NO será afectado.</p> <!-- No está dentro de un div -->

Selector Hijo Directo (Child Selector)

Selecciona un elemento que es **hijo directo** de otro elemento (no descendientes anidados más profundos).

Se denota con el símbolo mayor que >.

ul > li {
  /* Selecciona todos los elementos li que son hijos directos de un ul */
  border-bottom: 1px dashed #ccc;
}

HTML:

<ul>
  <li>Ítem 1</li> <!-- Afectado -->
  <li>Ítem 2</li> <!-- Afectado -->
  <li>
    <ul>
      <li>Sub-ítem</li> <!-- NO afectado por el selector `ul > li` -->
    </ul>
  </li>
</ul>

Selector de Clase Múltiple

Selecciona un elemento que tiene **TODAS** las clases especificadas.

Se denota encadenando los nombres de clase con puntos, sin espacios.

.btn.activo {
  /* Selecciona un elemento que tiene la clase 'btn' Y la clase 'activo' */
  background-color: green;
}

HTML:

<button class="btn">Botón Normal</button> <!-- NO afectado -->
<button class="btn activo">Botón Activo</button> <!-- Afectado -->
<button class="activo">Solo Activo</button> <!-- NO afectado -->

Estos son solo los selectores más básicos, pero dominar su combinación te permitirá apuntar a casi cualquier elemento en tu HTML para darle estilo.
