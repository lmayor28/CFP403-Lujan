Intro a la Manipulación del DOM con JavaScript

¿Qué es el DOM?

El **DOM (Document Object Model)** es una interfaz de programación para documentos HTML, XML y SVG. Representa la página como una estructura de árbol donde cada nodo es un objeto (un elemento HTML, un atributo, un texto, etc.).

JavaScript puede usar el DOM para **acceder y modificar** el contenido, la estructura y los estilos de una página web después de que se ha cargado en el navegador. Esto es lo que hace que las páginas web sean interactivas y dinámicas.

Seleccionar Elementos (La Base)

Antes de modificar algo, necesitas seleccionar el elemento HTML que te interesa.

  • Seleccionar por ID (si el elemento tiene un atributo id único):
    • const miElemento = document.getElementById('miId');
  • Seleccionar el primer elemento que coincide con un selector CSS (más flexible):
    • const primerParrafo = document.querySelector('p');
const primerElementoConClase = document.querySelector('.miClase');
  • Seleccionar TODOS los elementos que coinciden con un selector CSS (devuelve una lista - NodeList):
    • const todosLosParrafos = document.querySelectorAll('p');
const todosLosElementosConClase = document.querySelectorAll('.miClase');

Modificar Contenido

Una vez que tienes una referencia a un elemento, puedes cambiar su contenido.

  • Cambiar el texto (ignora etiquetas HTML dentro):
    • miElemento.innerText = 'Nuevo texto simple';
    miElemento.textContent = 'Otro nuevo texto simple';
  • Cambiar el contenido HTML (puede incluir etiquetas HTML):
    • miElemento.innerHTML = '<strong>Texto en negrita</strong> y más';

    ¡Cuidado con innerHTML! No insertes texto directamente desde entradas de usuario sin validación por seguridad.

Modificar Atributos

Cambiar atributos como src, href, class, etc.

const imagen = document.getElementById('miImagen');
imagen.src = 'nueva-imagen.jpg';
imagen.setAttribute('alt', 'Nueva descripción');
imagen.classList.add('borde-rojo'); // Añade una clase CSS

Modificar Estilos (CSS)

Cambiar propiedades CSS directamente. Las propiedades con guiones (`background-color`) se escriben en camelCase (`backgroundColor`).

miElemento.style.color = 'red';
miElemento.style.backgroundColor = 'yellow';
miElemento.style.display = 'none'; // Oculta el elemento

Aunque es posible, para cambios de estilo complejos o múltiples, es mejor añadir/eliminar/alternar clases CSS (usando classList) en lugar de modificar muchos estilos individualmente.

Estos son solo los primeros pasos en la manipulación del DOM con JavaScript. La combinación de seleccionar elementos y modificar sus propiedades (`innerText`, `innerHTML`, atributos, `style`) te abre un mundo de posibilidades para hacer tus páginas dinámicas.
