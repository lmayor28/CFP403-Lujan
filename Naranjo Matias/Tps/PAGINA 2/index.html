Estructura y Elementos HTML

Este ejemplo demustra como utlilizar las etiquetas basicas de HTML. Aqui encontraras elementos como encabezados, parrafos, listas, tablas y formularos

Etiquetas y Conceptos:

  • Encabezados: <h1>, <h2>, <h3> etc.
  • Parrafos: <p>
  • Listas: <ol> y <ul>
  • Tablas: <table>, <tr>, <th>, <td>
  • Formularios: <form>, <input>, <textarea>, <button>
  • Estructura Semantica: <header>, <nav>, <section>, <aside>, <footer>
